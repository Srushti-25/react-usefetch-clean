# 📸 React useFetch Project

## 📌 Project Overview
This project is built using React and demonstrates how to fetch data from an API using a custom hook (`useFetch`).  

The application displays photo data in a grid layout using reusable components.  
Instead of showing actual images, colored boxes are used to represent each item for a cleaner UI.

---

## 🚀 Features
- Custom React Hook (`useFetch`)
- API data fetching
- Loading and error handling
- Reusable `DataGrid` component
- Responsive grid layout
- Dynamic colored cards using Tailwind CSS

---

## 🛠️ Technologies Used
- React (Vite)
- JavaScript (ES6)
- Tailwind CSS
- JSONPlaceholder API
